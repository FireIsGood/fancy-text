Fancy Text

A converter from boring normal text to fancy Unicode text.

This uses various Unicode Blocks such as the mathematical symbols. For more information, go check out Unicode Explorer, more specifically most of these come from the Mathematical Alphanumeric Symbols section. Some symbols like "e" are marked as "reserved" in the unicode block when they already have a symbol from elsewhere.
