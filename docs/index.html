Fancy Text

A converter from boring normal text to fancy Unicode text.

This uses various Unicode Blocks such as the mathematical symbols. For more information, go check out Unicode Explorer, more specifically most of these come from the Mathematical Alphanumeric Symbols section.

Some symbols are marked as "reserved" in their unicode block when they already have a symbol from elsewhere. For these, I followed the replacements through to find what the Unicode website said was related. If there are weird discrepencies, that's probably why.
